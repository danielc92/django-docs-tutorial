Welcome to Daniels Poll App

This application was built using Django 2.2 and Python 3.7 in Ubuntu 19.04. I followed the official Django documentation and made some adaptations along the way. I plan to further improve my web development knowledge and refining my ablities in Django..

Take me to the questions list

Concepts

Below are the concepts covered in building the polling application

  • Creating and using virtual environments
  • Django settings configuration
  • Dynamic routing
  • Custom templates
  • Using the Django Admin Panel to update, create, delete records
  • Posting Forms
  • Creating ORM models and creating database tables from them
  • Querying database using Django ORM models
  • Deploying Django Application to Cloud Infrastructure as a service (Pythonanywhere)

Features

This application allows users to:

  • Create questions
  • Create choices for questions
  • Vote for a given question based on question id
  • View results for a given question based on question id
  • View complete list of user submitted questions
