Welcome to Daniels Poll App

This application was built using Django 2.2.1 and Python 3.7 in Ubuntu 19.04. I followed the official Django documentation and made some adaptations along the way. I plan to further improve my web development knowledge and refining my ablities in Django.

Concepts

Below are the concepts covered in building the polling application

  • Python virtual environments
  • Django project configuration
  • Django administration
  • Dynamic routing
  • Template engines
  • HTTP/HTTPS methods
  • Relational databases
  • CRUD operations
  • ORM models
  • Django deployment to the Cloud
  • Responsive web design
  • Application security

Features

This application allows users to:

  • Create questions
  • Create choices for questions
  • Vote for a given question based on question id
  • View results for a given question based on question id
  • View complete list of user submitted questions
