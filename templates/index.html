Welcome to Daniel Corcoran's Poll App

This is the first application I built using Django web framework. The official Django documentation was used as a starting point, following that, I adapted the application with more features, including the integration of an open source charting framework. It posed some challenges, in learning the overall structure of Django as well as becoming familiar with the way Django implements database ORM models, and deployment.

View polls

Create and explore polls

Satisfy your curiosity. Create polls for the world to interact with. Explore a wide range of polls of various categories, created by the community.

Create voting options and vote

Once your poll have been created and posted to the database, you can add choices and have the community vote, bringing you the answer you need.

Visualize

View insightful, elegant, animated visualizations relating to community polls. Charts are generated using open source javascript library Chart.js

Concepts

Below are the concepts covered in building the polling application

  • Containing dependencies in virtual environment and version control.
  • Routing, HTTP methods, Template rendering with Jinja
  • Database design, mapping and CRUD operations
  • Django deployment to the Cloud
  • Responsive web design
  • Security considerations; hashing, environment variables, csrf tokens.
